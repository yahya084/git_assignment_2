\# GitHub Features



1\. Repositories – Store and manage code.

2\. Branches – Work on features separately.

3\. Pull Requests – Review and merge changes.

4\. Issues – Track bugs and tasks.

5\. Actions – Automate workflows (CI/CD).

6\. Projects – Organize tasks using boards.

7\. Wiki – Documentation pages for projects.

8\. Security Alerts – Detect vulnerabilities.

9\. Forking – Copy repositories to your account.
